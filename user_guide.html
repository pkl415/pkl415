Panduan Pengguna

Halaman ini berisi panduan singkat untuk menggunakan aplikasi Inventaris Barang. Ikuti langkah-langkah berikut untuk memanfaatkan fitur yang tersedia.

📦 Tambah Barang

Gunakan menu Tambah Barang untuk menambahkan barang baru ke dalam sistem:

  • Isi nama barang, kategori, dan jumlah.
  • Klik tombol Tambah untuk menyimpan data.

✏️ Edit Barang

Jika terdapat perubahan data barang, gunakan menu Edit Barang:

  • Masukkan ID barang yang ingin diperbarui.
  • Ubah nama, kategori, atau jumlah sesuai kebutuhan.
  • Klik tombol Simpan untuk memperbarui data.

📑 Pinjam & Riwayat

Untuk peminjaman barang:

  • Pilih menu Pinjam Barang, lalu isi nama peminjam, barang, jumlah, dan tanggal kembali.
  • Cek catatan peminjaman pada menu Riwayat untuk melacak aktivitas peminjaman barang.

🗂️ Kategori & Data Barang

Gunakan menu Kategori untuk mengelola kategori barang. Data barang yang sudah tercatat dapat dilihat secara lengkap di menu Data Barang.

🔍 QR Code

Setiap barang dilengkapi dengan kode unik (QR Code):

  • Gunakan menu QR Code untuk men-generate dan mencetak QR Code barang.
  • QR Code mempermudah pencarian dan pendataan barang secara cepat.

ℹ️ Tips Penggunaan

  • Pastikan selalu mengecek data sebelum menyimpan.
  • Gunakan kategori agar data lebih terorganisir.
  • Cetak QR Code dan tempelkan di barang untuk inventarisasi yang lebih mudah.
